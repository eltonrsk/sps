# Student Pickup System (QR-Based) by Elton Ricardo

### 🛡️ Securing School Dismissal with Automated Authentication
This system replaces traditional, manual student pickup logs with a secure, QR-code-based verification process. It ensures that only authorized guardians can pick up students, streamlining the dismissal process for primary schools.

---

## 🚀 Key Features
* **QR Authentication:** Unique encrypted QR codes for every student/guardian pair.
* **Real-time Logs:** Instant digital record of pickup times and authorized personnel.
* **Security First:** Reduces the risk of unauthorized pickups through automated verification.
* **User-Friendly Interface:** Designed for quick use by school gate security and staff.

## 📋 Methodology
This project follows an **Agile Methodology**, allowing for iterative development and testing over a 34-week timeline. 

## 🔧 Installation & Setup
1. Clone the repository:
   `git clone https://github.com/cannonyele-dev/sps.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the application:
   `npm install sps`
